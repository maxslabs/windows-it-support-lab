# Windows IT Support Lab – Troubleshooting Scenarios

## Overview
This section documents practical troubleshooting exercises completed within a Windows IT Support Lab environment.

The purpose of these scenarios is to simulate real-world Service Desk and IT Support tasks commonly encountered in entry-level IT roles, including network diagnosis, system inspection, and basic Windows administration.

---

# Scenario 1 – User Unable to Access the Internet

## Issue Description
A user reports that their computer is unable to access the internet.

---

## Objective
Identify whether the issue is related to:

- IP configuration  
- Network connectivity  
- DNS resolution  
- Network adapter configuration
