# Windows IT Support Lab

## Overview

This project is a hands-on Windows IT support lab created using VirtualBox and Windows 10.

The lab is designed to develop practical troubleshooting, networking, and system administration skills aligned with entry-level IT Support and Service Desk roles.

It focuses on:
- Windows troubleshooting
- Networking diagnostics
- Administrative tools
- Structured troubleshooting methodology

---

## Lab Environment

| Component | Details |
|---|---|
| Virtualisation Platform | VirtualBox |
| Operating System | Windows 10 |
| Network Configuration | NAT |
| Host Machine | Personal Computer |

---

## Objectives

- Develop practical Windows troubleshooting skills
- Simulate real-world IT support tasks
- Practice networking and diagnostics commands
- Explore Windows administrative tools
- Build structured troubleshooting workflows

---

## Tools & Commands Used

### Windows Administrative Tools

- Task Manager
- Event Viewer
- Services Manager (`services.msc`)
- System Information

---

### Networking & Diagnostics Commands

#### `ipconfig`

Used to review:
- IP addressing
- Adapter configuration
- Gateway information

---

#### `ping`

Used to:
- Test network connectivity
- Verify internet access

---

#### `hostname`

Used to:
- Identify the device name
- Confirm system identity within a network

---

#### `systeminfo`

Used to:
- Review operating system information
- Inspect hardware and system configuration
- Assist with diagnostics
